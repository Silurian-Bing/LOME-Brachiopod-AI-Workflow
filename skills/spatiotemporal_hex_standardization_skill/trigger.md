# Trigger Conditions

Call this Skill when:

- palaeogeographic data must be converted into comparable spatial units;
- diversity or hotspot analysis depends on spatial sampling;
- PyRate input tables require standardized time-space records.

Do not call this Skill when:

- palaeocoordinates have not been validated;
- grid choice is still scientifically disputed;
- unpublished spatial details should not be exposed.
