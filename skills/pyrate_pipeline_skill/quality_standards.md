# Quality Standards

The Skill passes only if:

- runtime paths are checked;
- parameters are recorded;
- each replicate output is listed;
- convergence diagnostics are generated;
- failed replicates are not hidden.

Failure conditions:

- MCMC logs are missing;
- burn-in or sampling frequency is undocumented;
- PyRate input was generated from unreviewed data;
- convergence is not checked.
