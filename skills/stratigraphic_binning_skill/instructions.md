# Instructions

1. Preserve `early_interval`, `late_interval`, `max_ma`, and `min_ma`.
2. Fill missing late interval from early interval only when this rule is declared.
3. Use GTS2020 or a named time scale.
4. Assign bins based on overlap between occurrence age range and bin boundaries.
5. Report records spanning multiple bins.
6. Do not narrow an age range without evidence.
