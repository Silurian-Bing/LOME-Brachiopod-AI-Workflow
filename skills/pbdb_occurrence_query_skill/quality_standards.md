# Quality Standards

The Skill passes only if:

- query parameters are recorded;
- raw records are saved before modification;
- record count is reported;
- all PBDB IDs and reference fields are preserved when available;
- retrieval date and software/package versions are logged.

Failure conditions:

- PBDB query fails silently;
- records are filtered without a filter report;
- original taxon names are overwritten;
- output file has no metadata.
