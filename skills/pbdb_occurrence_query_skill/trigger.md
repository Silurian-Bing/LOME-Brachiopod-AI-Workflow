# Trigger Conditions

Call this Skill when:

- the workflow requires fresh PBDB occurrence records;
- a database snapshot must be documented;
- PBDB records need to be compared with GBDB or local curated records.

Do not call this Skill when:

- records have already been frozen for the run;
- the task is taxonomy cleaning, stratigraphic binning, or palaeocoordinate reconstruction;
- PBDB network access is unavailable and no local snapshot is specified.
