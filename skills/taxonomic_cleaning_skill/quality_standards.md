# Quality Standards

The Skill passes only if:

- original names remain intact;
- every changed name has a decision reason;
- uncertain records are flagged;
- excluded records are listed;
- human review is requested for decisions that alter key results.

Failure conditions:

- automatic synonymy is undocumented;
- `cf.`/`aff.` records are silently treated as confirmed;
- genus and species-level records are mixed without rule documentation.
